# CurbUI 用户服务协议

**版本号：2026-v1**
**生效日期：2026年4月10日**

感谢您关注并使用 **CurbUI**。本协议旨在明确您在使用本组件库及相关技术资源时的权利与义务。

### 一、 许可说明

1. **代码开源**：CurbUI 核心源代码以 **MIT 许可证** 形式公开发布。您拥有学习、修改以及将其用于商业项目的自由，但须遵守 MIT 协议的相关署名义务。
2. **文档与演示**：本团队免费提供配套的 `curb-ui-docs` 示例文档及基于 Vercel 部署的预览站点。
3. **技术咨询**：针对“项目咨询”板块，现阶段主要定位为公益性技术交流。**本团队不就相关咨询收取任何费用，并将在能力范围内尽力确保回复的及时性，但不针对具体的响应时间或回复频率做出法律意义上的承诺。**

### 二、 知识产权声明

1. **视觉标识**：CurbUI 品牌 Logo（抽象交叠“CU”标识）、品牌名称及特定 UI 视觉设计语言等资产的知识产权均归属本团队所有。
2. **贡献者权益**：若您为项目提交了 Pull Request (PR) 并被采纳，则视为您同意该部分代码按照 MIT 协议授予全球用户使用。

### 三、 技术咨询与交流规范

1. 用户可通过 GitHub Issues、Gitee 评论或官方指定的渠道发起“项目咨询”。
2. **建议性质**：所有回复内容、方案建议或提供的代码片段仅供参考，不构成任何形式的交付承诺、法律合同或服务保障。
3. **技术局限性**：由于跨端环境（如 Android、iOS、Web等）的复杂性，本团队不保证提供的建议为特定环境下的唯一或最优解决方案。

### 四、 免责条款（重点）

1. **按现状提供**：**CurbUI 按“现状（As-Is）”提供，不附带任何形式的明示或暗示担保。** 用户需自行承担在生产环境中使用本库所带来的风险，包括但不限于 UI 兼容性异常、性能波动或系统冲突。
2. **损失限制**：在任何情况下，本团队均不对因使用 CurbUI 或参考相关建议而导致的直接、间接、偶然或特殊的利润损失、数据丢失或业务中断承担赔偿责任。
3. **外部依赖**：对于因 Vercel 部署、Gitee 同步延迟或 CDN 服务故障等第三方基础设施问题导致的访问异常，本团队不承担补偿责任。

### 五、 社区文明规范

1. **开源互助**：请尊重开源精神，严禁利用 CurbUI 资源从事违法、侵权、散布恶意代码或干扰他人业务的行为。
2. **品牌回馈**：在商业项目中使用时，虽然不作强制要求，但我们建议您在项目致谢页保留“Powered by CurbUI”字样，您的支持是我们持续优化的动力。

### 六、 其他

1. **协议修订**：本团队保留随项目发展修订本协议的权利，修订内容一经在仓库 README 或官网公示即产生效力。
2. **条款可分性**：如本协议部分条款因法律原因被认定为无效，不影响其他条款的继续执行。
