# CurbUI 隐私政策

**版本号：2026-v1** **生效日期：2026年4月10日**

本隐私政策旨在向您说明 **CurbUI**（以下简称“我们”）如何处理您在访问官网、查看文档或参与项目交互时涉及的信息。我们深知个人隐私的重要性，并承诺采取行业标准的安全措施保护您的信息。

### 一、 特别说明：关于 CurbUI 示例环境（App & Web）
**为保障您的知情权，请在使用 CurbUI 演示应用前了解以下数据处理逻辑：**
1. **示例模块**：除“项目咨询”和“帮助反馈”外，本应用中展示的其他功能模块（如资产管理、个人中心等）目前**均未对接后端 API，仅作为 UI 交互示例使用**。这些模块不会收集、存储或上传您的任何真实数据。
2. **本地化体验**：示例模块中的所有数据操作均在您的设备本地完成，关闭应用后数据即刻消失。

### 二、 我们收集的信息
在您与我们互动的过程中，仅在以下真实生效的场景中会涉及数据交互：

1. **项目咨询与帮助反馈**：当您主动通过这两个模块发起联系时，我们会收集您填写的 **联系方式（如邮箱/微信号）** 以及您提交的 **反馈内容、截图或项目描述**。这些信息仅用于我们与您取得联系并解决问题。
2. **访问日志**：当您访问我们的 Vercel 预览站点或文档时，宿主平台（Vercel）可能会自动记录您的 IP 地址、浏览器类型等基础网络日志，用于站点性能优化。
3. **开源平台交互**：当您在 GitHub/Gitee 提交 Issue 或 PR 时，您的公开个人资料（用户名、头像等）由该平台处理并展示。

### 三、 信息的使用方式
1. **回复与支持**：利用您提供的联系信息，针对“项目咨询”和“帮助反馈”内容提供技术支持或合作洽谈。
2. **服务优化**：通过分析匿名化的访问统计数据，提升 CurbUI 组件库的加载速度及文档体验。
3. **安全保障**：用于检测、预防或处理欺诈及安全技术问题。

### 四、 信息的共享与保护
1. **绝不买卖**：我们承诺绝不将您的联系方式或反馈内容出售、出租或转让给任何第三方。
2. **第三方宿主**：我们的站点部署在 Vercel 平台，数据处理受其隐私政策约束。
3. **咨询保密性**：对于“项目咨询”中涉及的非公开业务逻辑，我们内部严格保密，仅限核心开发成员阅览。

### 五、 您的权利
您对自己的个人信息拥有以下权利：
* 您可以随时要求我们删除已记录的联系信息及沟通记录。
* 您可以要求我们停止向您发送技术回复或更新通知。
* 如需行使上述权利，请通过 GitHub Issue 或本协议公示的联系方式与我们取得联系。

### 六、 免责声明
1. **演示风险**：由于示例模块未对接 API，请勿在示例输入框中填写任何真实、敏感的个人隐私数据（如密码、银行账号等）。因用户在示例模块填写敏感信息导致的泄露风险，本团队不承担相关责任。
2. **外部链接**：本应用或文档可能包含指向第三方网站的链接，我们不对第三方的隐私实践承担责任。

### 七、 政策修订
我们保留随时修改本隐私政策的权利。修订后的政策将即时在官网或仓库公示。**若您在修订后继续使用 CurbUI，即视为您同意修订后的隐私政策。**
